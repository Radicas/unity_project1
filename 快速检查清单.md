# Unity项目环境搭建 - 快速检查清单

## 📝 使用说明
按照顺序逐项检查，每完成一项打勾 ✅

---

## 阶段1：准备工作

- [ ] **检查Windows版本**
  - Windows 10/11 (64位)
  - 至少 8GB 内存
  - 至少 20GB 可用磁盘空间

- [ ] **准备网络环境**
  - 稳定的网络连接
  - 如需，配置VPN/代理

---

## 阶段2：安装Unity Hub

- [ ] **下载Unity Hub**
  - 访问：https://unity.com/download
  - 下载 UnityHubSetup.exe

- [ ] **安装Unity Hub**
  - 运行安装程序
  - 完成安装向导

- [ ] **登录Unity账号**
  - 创建/登录账号（免费即可）
  - 验证登录状态

---

## 阶段3：安装Unity编辑器

- [ ] **添加Unity版本**
  - 在Unity Hub中点击 "Install Editor"
  - 选择版本：**2022.3.34f1c1**

- [ ] **选择安装模块**
  - ✅ Microsoft Visual Studio Community 2022
  - ✅ Windows Build Support (IL2CPP)
  - ✅ Windows Build Support (Mono)
  - ✅ Android Build Support（可选但推荐）
  - ✅ Android SDK & NDK Tools（可选）
  - ✅ OpenJDK（可选）

- [ ] **完成安装**
  - 等待下载完成（30分钟-2小时）
  - 确认安装成功（显示绿色对勾）

---

## 阶段4：打开项目

- [ ] **添加项目到Unity Hub**
  - 点击 "Projects" → "Add"
  - 选择项目文件夹：`C:\Repos\unity_demo`

- [ ] **打开项目**
  - 使用Unity 2022.3.34f1c1打开
  - 等待资源导入完成（5-15分钟）

---

## 阶段5：验证配置

- [ ] **检查Console窗口**
  - 无红色错误（Errors）
  - 黄色警告可忽略

- [ ] **检查Package Manager**
  - Window → Package Manager
  - 确认关键包已安装：
    - Unity Ads
    - Unity Analytics
    - TextMesh Pro

- [ ] **测试运行项目**
  - 打开 MainMenu.unity 场景
  - 点击播放按钮 ▶️
  - 游戏正常运行

---

## 阶段6：构建测试

- [ ] **配置构建设置**
  - File → Build Settings
  - 选择 "PC, Mac & Linux Standalone"
  - 切换平台（如需要）

- [ ] **执行构建**
  - 点击 "Build" 按钮
  - 选择输出文件夹
  - 等待构建完成（5-20分钟）

- [ ] **测试构建的游戏**
  - 运行生成的 .exe 文件
  - 确认游戏功能正常

---

## 阶段7：环境验证

- [ ] **Unity版本正确**
  - 确认使用 2022.3.34f1c1

- [ ] **项目无错误**
  - Console窗口无红色错误
  - 所有资源正常加载

- [ ] **可以正常开发**
  - 可以编辑场景
  - 可以修改代码
  - 可以运行测试

---

## 🎯 完成标志

如果以上所有项目都已完成并打勾，说明：
- ✅ 开发环境已成功搭建
- ✅ 可以开始开发工作
- ✅ 可以构建发布版本
